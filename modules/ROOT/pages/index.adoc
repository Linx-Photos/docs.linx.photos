= Linx Photos Documentation
:navtitle: Introduction

Welcome to the official documentation for **Linx Photos**, the "Zero-Config" SaaS for professional photographers.

Linx Photos goes beyond simple image hosting. It is an end-to-end ecosystem designed to handle high-resolution asset delivery, legally binding contract automation, and seamless integration into any photographer's existing website.

== Core Features

* **High-Speed CDN Galleries**: Optimized for professional portfolios.
* **RAW File Commerce**: Monetize high-end work with secure checkout and usage agreements.
* **Legal Engine**: Automated contract handling with OCR capabilities.
* **Zero-Config Integration**: "Magic" style adoption for host websites via the single-script loader.
